# Neues Kundenprojekt anlegen

1. Ordner unter `hq/03_Kundenprojekte/{Slug}/` erstellen (wie in `_registry.json`).
2. Dateien kopieren aus dieser Vorlage (alle `.md` im Ordner).
3. Eintrag in `03_Kundenprojekte/_registry.json` ergänzen.
4. Optional: `projects/{bot_project_slug}/` im Bot-Repo anlegen (siehe `projects/README.md`).
