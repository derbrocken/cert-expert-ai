# Cert-Expert — Brand-Spec (für Cursor / UI)

**Quelle:** offizielle Brand-Guide-SVG (von Mark, 2026-06-07). Logo-Dateien in diesem Ordner: `Logo.svg` (Mantis + Wortmarke „cert-expert"), `Mantis.svg` (nur Bildmarke).

## Offizielle Farben (HEX)
**Entscheidung Mark (2026-06-07): Firmenfarbe = Vermilion, darauf ausrichten. KEIN Violett/Indigo.**

| Rolle | HEX | Verwendung |
|-------|-----|-----------|
| **Vermilion (Leitfarbe)** | `#DA2102` | Logo + **UI-Akzent + CTAs/aktiv** |
| Wortmarke / Text | `#1A1C20` | Fließtext, Überschriften, dunkle Flächen |
| Hintergrund-Grau | `#F1F3F8` | Header, Sidebar, Flächen |
| Fläche | `#FFFFFF` | Karten/Panels |
| Alt-Welten (nur falls nötig) | Cobalt `#2A1BFF` · Violet `#3F00A8` · Indigo `#3A1BDA` | aktuell NICHT verwenden |

## UI-Regel
- **Vermilion `#DA2102` ist die Leitfarbe** (Logo, Buttons, aktive Tabs, Auswahl).
- Basis ruhig/hochwertig: Flächen `#F1F3F8`/`#FFFFFF`, Text `#1A1C20`.
- **Ampel braucht eine vom Vermilion klar unterscheidbare Statusfarbe** — im Design-Pass klären: „fehlt/Blocker" als tieferes **Bordó/Crimson** (z. B. `#B3261E`) statt Orange-Rot, plus Grün `#1D9E75` / Gelb `#E0A012`. (Detail für den späteren Design-Pass, nicht jetzt.)

> Hinweis: Das Mockup im Chat war in Indigo — **überholt**. Leitfarbe ist Vermilion.

## Logo-Einsatz
- Cursor: `Logo.svg` / `Mantis.svg` nach `apps/certification-os/public/brand/` legen und im Header (Slice 5) einbinden. Wortmarke dunkel (`#1A1C20`) auf hellem Header.
