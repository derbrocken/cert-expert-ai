Installation und Programmstart

Die Installation des Programms zu Gefährdungsbeurteilung ist nicht erforderlich. Die Programm-Funktionen stehen auch nach Kopieren der Daten auf eine Festpaltte oder einen mobilen Datenträger (z.B. USB-Stick) zur Verfügung. Dennoch bietet ein installiertes Programm einige Vorteile:

  • höhere Geschwindigkeit
  • gespeicherte Daten der Gefährdungsbeurteilung sind durch Doppelklick startfähig
  • alle erforderlichen Daten sind auf dem Arbeitsrechner jederzeit verfügbar
  • eine Versionskontrolle z.B. nach Updates ist möglich, neue Programmversionen können auf die gespeicherten Daten zugreifen

Nachteile einer Installation sind:

  • die Installation braucht Zeit (zwischen ca. 2 und 10 Minuten je nach Rechnerleistung)
  • der Anwender ist auf einen Rechner festgelegt
  • möglicherweise sind für die Installation besondere Nutzerrechte erforderlich

So können Sie installieren:

1. Packen Sie die Archivdatei (*.zip) aus.
2. Wenn das Programm automatisch startet, betätigen Sie die Schaltfläche "Installieren". Falls auf Ihrem Computer diese Funktion nicht aktiviert ist, wählen starten Sie die Datei AUTORUN.EXE" und wählen installieren.
3. Befolgen Sie die Aufforderungen des Installationsprogramms und warten Sie, bis das Programm die erfolgreiche Installation meldet.

Nach der Installation finden Sie im Startmenü Ihres Rechners unter der Rubrik "Programme" den neuen Eintrag "Gefährdungsbeurteilungen" bzw. den Namen, den Sie bei der Installation eingetragen hatten. In dieser Gruppe befindet sich das Start-Symbol für das Programm. Mit einem Klick auf dieses Symbol starten Sie das Programm.

Diese Hilfe-Datei steht Ihnen jederzeit zur Verfügung ([F1]-Taste). Zusätzlich können Sie mit Hilfe der rechten Maustaste die oft benötigten Programmfunktionen bequem aufrufen.

Systemvoraussetzungen

·  PC mit Windows 8/10/11
·  MS-Excel, MS-Word (für einige Sonderanwendungen)
·  bei Vollinstallation ca. 1,5 GB freier Speicherplatz
