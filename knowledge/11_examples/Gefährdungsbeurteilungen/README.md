# Beispiele — Gefährdungsbeurteilungen

Hier liegen **anonymisierte oder synthetische Beispielauszüge** für Gefährdungsbeurteilungen, die zeigen, wie formuliert und gegliedert werden kann.

Die Dateien dienen der **Orientierung für Kuratierung und Tests**, nicht als Vorlage für reale Aufträge ohne Prüfung.
