# Control Measures (`control_measures/`)

Dieser Ordner ist für **kuratierte Leitlinien zu Schutz- und Steuerungsmaßnahmen** vorgesehen — wie Maßnahmen typischerweise beschrieben, begründet und gegen Risiken abgegrenzt werden.

Inhaltlich ergänzt er die übrige `guides/`-Struktur; **konkrete Auftragsszenarien** bleiben außen vor.
