# Event Phases (`event_phases/`)

Dieser Ordner dient **Phasenmodellen und zeitlichen Ablaufmustern** für Veranstaltungen und vergleichbare Einsatzsituationen (z. B. Planung, Durchführung, Nachbereitung).

Es werden **strukturelle Orientierungshilfen** abgelegt, keine Einsatzpläne für einzelne Objekte oder Termine.
