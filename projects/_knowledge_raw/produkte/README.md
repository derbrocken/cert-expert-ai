# Rohwissen — Dokumentprodukte / Methodik (global)

Rohmaterial zur **Methodik** der Cert-Expert-Produkte: Tiefe, Kapitelstruktur,
Formulierungsstandards, Reviewer-Erwartungen — **unabhängig** von einer
einzelnen SDL.

SDL-spezifische Beispiel-Dokumente gehören nach `../sdls/<sdl>/..._specific/`.

**Kuratierungsziel:** `knowledge/products/<produkt>/` und ggf. `knowledge/guides/`.
