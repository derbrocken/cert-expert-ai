# Rohwissen — Schnittstellen (global, querschnittlich)

Partner- und Kooperationslogik, die **mehrere SDLs** und **mehrere Produkte**
(GB, SK, EC/ODA, Einweisungen) betrifft.

**Nicht** unter `sdls/.../subtypes/` ablegen.

| Ordner | Hinweis |
|--------|---------|
| `sanitaetsdienst/` | Kein SDL-Subtyp |
| `polizei/`, `feuerwehr/` | Hoheit / Rettung |
| `kunde_auftraggeber/` | Auftraggeber, Veranstalter, Informationspfade |

Erweiterungen (z. B. Hallenbetrieb, Technik) bei Bedarf analog anlegen.

**Kuratierungsziel:** SDL-`base.md`, Produktregeln, künftige dedizierte Interface-Module unter `knowledge/`.
